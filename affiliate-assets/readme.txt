== Description ==

AffiliateAssets es un sistema completo de gestión de afiliados para WordPress que transforma tu sitio en una plataforma de inversión y referidos.

**Características Principales:**

* **Sistema de Afiliados Completo** - Registro automático, estados múltiples, aprobación manual o automática
* **Tracking Avanzado** - Cookies persistentes, URLs amigables, registro detallado de visitas
* **Códigos QR Personalizables** - Generación automática, descarga en PNG/SVG, tracking de escaneos
* **Panel de Administración** - Dashboard con estadísticas, gestión de afiliados, configuración completa
* **Panel del Afiliado** - Tabs intuitivos, enlaces copiables, deep linking, compartir en redes

**Inspirado en SliceWP:**
Este plugin sigue los patrones arquitectónicos y de UX/UI de SliceWP (https://github.com/WordPressBugBounty/plugins-slicewp), implementando las mejores prácticas observadas en su estructura de código, sistema de tracking y organización del panel de administración.

== Installation ==

1. Sube la carpeta `affiliate-assets` a `/wp-content/plugins/`
2. Activa el plugin desde el menú 'Plugins' en WordPress
3. Ve a AffiliateAssets → Configuración para configurar el plugin
4. Crea una página con el shortcode `[aa_affiliate_area]` para el panel de afiliados

== Frequently Asked Questions ==

= ¿Requiere WooCommerce? =
Sí, AffiliateAssets requiere WooCommerce 5.0 o superior para funcionar correctamente.

= ¿Cómo se registra un afiliado? =
Los usuarios se registran automáticamente como afiliados al comprar el producto de membresía configurado ($500 por defecto).

= ¿Puedo aprobar afiliados manualmente? =
Sí, puedes configurar aprobación manual o automática desde la configuración del plugin.

= ¿Cómo funciona el tracking? =
El plugin usa cookies con duración configurable (default 30 días) para trackear visitantes referidos.
